### Power Query Shortcuts

**1. Zoom Controls:**
   - Zoom In: `Ctrl + Shift + +`
   - Reset Zoom to 100%: `Ctrl + 0`

**2. Code Navigation and Editing:**
   - Move Code Up: `Alt + Up Arrow`
   - Move Code Down: `Alt + Down Arrow`
   - Duplicate Code: `Alt + Shift + Down Arrow`
   - Scroll Screen Up: `Ctrl + Up Arrow`
   - Scroll Screen Down: `Ctrl + Down Arrow`

**3. Code Formatting:**
   - Increase Indentation: `Ctrl + ]`
   - Decrease Indentation: `Ctrl + [`

**4. Commenting Code:**
   - Comment or Uncomment Line: `Ctrl + /`
   - Toggle Block Comment (Multiline Comment): `Alt + Shift + A`

**5. Code Deletion:**
   - Delete Code Line: `Ctrl + Shift + K`

**6. Editor Controls:**
   - Open Advanced Editor: `Alt + 1`

**7. Text Selection and Editing:**
   - Go to Next Highlighted Text: `F7`
   - Go to Previous Highlighted Text: `Shift + F7`
   - Select Multiple Text Input Locations: `Alt + Click`

These shortcuts provide efficient ways to manage and manipulate your code in Power Query, enhancing productivity and ease of use.
